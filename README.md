# 🎧 B&O Landing Page

## 📄 Project Description

A responsive landing page for **Bang & Olufsen**, featuring product showcases, categories, company description, and a contact form.
The project is built with semantic HTML, SCSS using BEM methodology, and follows a mobile‑first responsive design.

## 🎨 Design Mockup

🔗 [Figma Layout](https://www.figma.com/design/DtkQmQ797hk0nI4KfMi2Uq/BOSE-New-Version?node-id=6817-212&t=cVEhyvZIYXtMqQCM-0)

---

## 🛠️ Technologies Used

- **HTML5** – semantic structure
- **SCSS (CSS3)** – variables, mixins, modular structure (BEM)
- **Flexbox & CSS Grid** – layout system
- **Media Queries** – responsive design (mobile‑first)
- **Git & GitHub Pages** – version control and deployment
